RUASOL MARKET v3.1 ULTIMATE CUSTOM UI
======================================

Bu sürüm v3 custom UI üzerine kalite ve güvenlik revizyonudur.

Eklenen / güçlendirilenler:
- Chest GUI yok. /pazar tamamen custom client screen açar.
- Ayrı texture seti: ana pazar, müzayede, depo, tablo, admin paneli.
- Button hover / normal / pasif texture state.
- Icon atlas: RC, Pazar Kredisi, mühür/kategori/depoya uygun sembol dili.
- UI içinde riskli işlemler için onay modalı.
- İlan listesinde gerçek NBT preview: client tarafı Base64 NBT'den ItemStack decode eder.
- Tooltip: item üzerine gelince gerçek item tooltip'i gösterilir.
- Açık artırma sekmeleri: tümü, RC, Pazar Kredisi, son dakika, teklifim.
- Açık artırma birimi: RC veya Pazar Kredisi.
- Pazar Kredisi ile normal sabit fiyatlı satış yok; yalnızca açık artırma birimi olarak kullanılır.
- Tablo sekmeleri: kazanç, harcama, satış, alış, beğeni, müzayede fatihleri.
- Client action rate limit: spam packet yükünü azaltır.
- WAL dosyası: world/serverconfig/ruasol_market_wal.jsonl.
- allow_internal_test_economy varsayılan false: production güvenliği artırıldı.
- Ürün oluşturma ve teklif kayıt akışlarında save başarısızlığına karşı rollback/iade denemeleri güçlendirildi.
- Süresi dolan sabit ilanlarda satıcı online ve envanteri uygunsa direkt teslim denemesi yapılır, değilse depo.

Bilinçli sınırlar:
- Gerçek build sandbox ortamında alınamadı; Gradle/Forge dependency yok.
- UI texture'ları programatik pixel/dark fantasy asset olarak üretildi. Sonraki estetik aşamada elle çizilmiş daha yüksek kalite assetlerle değiştirilebilir.
- WAL recovery okuyucusu yönetimsel kayıt/inceleme için hazırdır; otomatik full transaction replay sistemi sonraki güvenlik aşamasında daha da büyütülebilir.
