RUASOL MARKET v3.6.4 LAYOUT + ECONOMY HOTFIX

Bu hotfix iki ana problemi kapatır:

1) UI / layout overhaul
- Panel tam 512x256 genişliğe taşındı.
- Tüm ana panel texture'ları yeniden üretildi; içerik çerçevesi artık sağda kesilmiyor.
- Sol menü yeniden hizalandı; OP Mührü alt dekorla ve Rehber ile çakışmaz.
- Kadim Pazar kategori/sıralama/arama satırları ayrıldı.
- Arama kutusu ve Ara butonu artık sıralama butonlarıyla çakışmaz.
- Liste kartları ve Detay/Teklif hitbox'ları daha geniş panele göre taşındı.
- Önce/Sonra butonlarında yön göstergesi düzeltildi: Önce <, Sonra >.
- OP Yönetim ekranında butonlar ve transaction metinleri artık üst üste binmez.
- Rehber sekmeleri daha geniş grid'e alındı, metin alanı genişletildi.

2) Vault / Essentials economy bridge hardening
- Arclight hybrid ortamında Vault API sınıfını plugin classloader üzerinden de arar.
- Vault provider yoksa Essentials direct API fallback denenir.
- Admin Ekonomi Test artık force reconnect yapar ve net debug status döner.
- No economy provider mesajı yerine hangi aşamada takıldığı log/debug metni ile görülebilir.
