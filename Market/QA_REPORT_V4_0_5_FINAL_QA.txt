Ruasol Market v4.0.5 Final QA Report

Kapatılan kritikler:
1. Profilim başlık/buton çakışması giderildi; butonlar başlığın altına taşındı.
2. Ürün Ekle süre açıklamasının butonlara binmesi kaldırıldı; ana işlem butonu footer alanına alındı.
3. OP Mührü yönetim işlemleri ve son kayıtlar ayrı kutulara ayrıldı.
4. MY_BIDS başlığı ve liste akışı eklendi; header artık Müzayede Tekliflerim gösterir.
5. OFFERS / MY_OFFERS header başlıkları eklendi.
6. Profilim posta sayısı snapshot içinde gerçek notification count ile doldurulur.
7. Duyuru sistemi MarketAnnouncementPacket + client overlay ile serverdan tüm clientlara iletilir.
8. Rollback iade yolu overstack ana el koymaz; parçalı teslim/depo yolunu kullanır.
9. Depo claim / claim all / auto delivery eski overstack kayıtlarına karşı split-stack teslim kullanır.
10. dataVersion ve network protocol 405'e güncellendi.
11. Açık artırma ücret UI'si ve server mantığı tutarlı: 1-2 saat ücretsiz, 3/5 saat config bedeli.
12. İlan detayı input görünümü özel koyu/altın input kutusuyla tutarlı.

Statik kontroller:
- Tüm Java dosyalarında brace dengesi temiz.
- ELLIPSIZATION / placeholder yok.
- K... / Müza... gibi kötü kısaltılmış tablo sekmesi yok.
- setHeldItem overstack iade yolu kaldırıldı.
- addItemStackToInventory(item.copy()) eski overstack claim yollarında kalmadı.
- Zip kök yapısı doğrudan build dosyalarıyla hazırlanmıştır.

Not:
- Bu ortamda Gradle/Forge derleme çalıştırılamadı. Windows tarafında BUILD_MOD.bat ile gerçek compile alınmalıdır.
